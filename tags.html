Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Document Tags

Tag Purpose
HTML represent a root of an HTML document
head provides general information about the document
body represents a content of an HTML document
title defines a title of an HTML document
meta Represents metadata that cannot be representated by other HTML meta-related elements.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p Paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header webpage header represents introductory content
div The Content Division element is the generic container for flow content
ol Ordered list Include an ordered list of elements
ul Unordered list Include an unordered list of elements
li Item in a list Include a item in a list of elements
blockquote block quotation Show formatted quotation
br break enter line break

Inline Elements

Tag Name Purpose Example Result
span span define inline section in document <span>Call me Ishmael</span> Call me Ishmael
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people&alt;/strong> at least a billion people
a anchor make hyper link <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code Show text in code style <code>code</code> code
